CIS Apple iOS Profile Build Kit

This Build Kit provides prescriptive guidance for establishing a secure configuration 
posture for Apple iOS that relies on Mobile Configuration Profiles created with Apple Configurator 2.17. This guide was tested against Apple iOS 18.0. This Build Kit is designed only for recommendations that can be set using configuration profiles. To properly set the additional recommendations, please follow the guidance in the CIS Apple iOS 18 v1.0.0 Benchmark and use Apple Configurator to install. To obtain the latest version of the Benchmark or the Build Kit, please visit http://benchmarks.cisecurity.org. If you have questions,comments, or have identified ways to improve this Build Kit, please write us at feedback@cisecurity.org.

The profiles contained in this Build Kit were built to the specifications provided by 
Apple:

https://developer.apple.com/documentation/devicemanagement/profile-specific_payload_keys
https://developer.apple.com/business/documentation/Configuration-Profile-Reference.pdf
